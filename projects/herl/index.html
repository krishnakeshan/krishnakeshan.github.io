Herl - An application to create short links with dynamic destinations

Live At

  • herl.me

Summary

  • Backend built with Go and MongoDB
  • Frontend built with HTML, CSS, JS and Bootstrap
  • Authentication and analytics built with Firebase
  • SSL/TLS certificate provided by Let’s Encrypt
  • Hosted on Google Cloud Platform’s Compute Engine offering on the basic instance.
  • Goroutines allow concurrent handling of large numbers of client requests simultaneously. This coupled with Go’s speed result in a monster of a server.
  • Mongo’s interface to Go allow easy interaction with the database.

About

Herl is an application that lets you create short urls whose destinations can be changed easily.

I built this application after noticing many dead links on the Internet. These links were dead not because the content didn’t exist but because it had been moved elsewhere. Such problems can be mitigated with Herl. It was used at my college for creating links to online classes so the teachers and students didn’t have to acquire a new link everyday. Currently it has about 100 users.

Usage

  1. Head over to https://herl.me and create an account. If you already have an account, simply log in with your details.
  2. In the “Create Smart Link” section, enter the initial destination you want your link to point to. You can leave this blank if you want a placeholder link.
  3. Add a title and an optional description. This lets you locate your links on your dashboard once you have a lot of links.
  4. Hit “Create” and a new link will appear below. Press the “Copy” button to copy the link to your clipboard.

You can edit existing links by simply entering new details in the “Your Links” section and saving.
