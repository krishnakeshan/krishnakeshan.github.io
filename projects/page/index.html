Page - Turn your computer into a home server

GitHub:

  • Page-Android
  • page-python

Summary

  • Clean UI
  • Built on Android with Java
  • Leverages Architecture Components:
    • Room: Storage of defined models with the ability to serve live data. This keeps the application reactive and consistent without the need to manually update UI components when data changes.
    • LiveData: The above-mentioned realtime nature of the application is made possible mainly because of LiveData in conjunction with Room. LiveData keeps datastreams up-to-date and allows propagation of changes through multiple layers.
    • ViewModel: Now the de-facto standard of application architecture, MVVM is the primary influence behind the architecture of Page. It greatly simplifies interactions between multiple UI components and interactions with the database. It does all this while keeping concerns separate.
    • Navigation Components: Leverages the new Navigation Components to implement a single-activity architecture. The various sub-screens are fragments and the Navigation Component handles transitioning between them and updating auxillary UI elements (such as Toolbars, Back buttons, etc.) automatically.
  • Sockets: Sockets form the communication backbone of the application. They allow transfer of messages and files between the connected hosts. JSON is used for representing messages.
  • Executors: The various network operations and other long-running operations are handled by Executors with a fixed pool size. This is suitable for home environments where the number of devices interacting with each other is not too high. This keeps the UI performant and efficiently performs background operations.
  • Raw Threads: A ping service constantly queries the server to get its online status and this is implemented as a standalone thread as the overhead of Executors isn’t desired here. This allows the application to automatically detect server status and notify any listeners.
  • Custom 9-Patch Backgrounds: While the in-built UI components that come with Android are sufficient most of the time, in Page custom backgrounds have been used to enhance the UI and make it more suited to the application’s style.

About

Page is an application that lets you turn your computer into a home server so that your devices can communicate over LAN.

I built this application as there was always something on my laptop that I needed on my phone and vice versa. I wanted to experiment with socket programming and this felt like a suitable project to demonstrate those skills.

Usage

  1. Download the server implementation page-python onto the machine you want to turn into a server.
  2. Follow the instructions in the repository to start your server.
  3. Download Page-Android onto your Android device and follow in-app instructions to connect your device to your server.

In case of server restarts (network failures, power outages, etc.) the device can be configured to the new address right inside the Android application by visiting the Settings screen.
